# 美团外卖收入价格计算器

美团外卖定价计算辅助工具，可以通过外卖定价计算商户的实际收入，也可以通过设置预期收入计算商品的定价。

使用方法：

配置好Python运行环境，使用Python直接执行文件即可

控制台输入价格时可直接使用`+-*/`等计算符号

注意事项：

美团在不同地区使用的抽成方式不同，此处采用的是 `(商品原价+打包费)*固定百分比` 方式。请根据实际情况修改百分比或算法。

该算法只考虑单个商品定价，并且优惠方式仅考虑了减配送费、满减、神券，未考虑其它折扣方式，输出的结果仅供参考。
